📖 Introduction

Programming starts with communication.

In Python, we communicate with the computer using functions.
The simplest one is:

print()

It tells Python to display something on the screen.

💻 Example
print("Hello, Python!")

Output:

Hello, Python!
▶ Try it yourself

Change the text:

print("Hello, world!")

👉 Try replacing "world" with your name.

2. What will this print?

3. Which is correct?

A)

print(Hello)

B)

⚠ Common mistakes
print(Hello)

❌ Wrong → missing quotes

Correct:

print("Hello")
💡 Tip

In Python:

Text → must be in quotes "like this"
Numbers → no quotes 123
