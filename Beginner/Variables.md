📖 Introduction

Imagine you are talking to a computer, and you want it to remember something.

For example:

your name
your age
your favorite game

Instead of repeating this information every time, we store it in a variable.

A variable is like a labeled box where you can store data.

💻 Example
name = "Anna"
age = 20

print(name)
print(age)

Output:

Anna
20
🧠 What is happening?
name stores text → "Anna"
age stores a number → 20
print() shows the stored values
▶ Try it yourself

Change the values:

print(age)

⚠ Common mistakes
❌ Using quotes for numbers
age = "25"

This is text, not a number.

❌ Invalid variable names
my-name = "Anna"

Hyphens are not allowed.

Correct:

my_name = "Anna"
💡 Tip

Python naming style:

use lowercase
separate words with _

Example:

favorite_game = "Minecraft"
